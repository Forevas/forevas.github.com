Windows下使用MinGW编译FFmpeg

  • FFmpeg
  • MinGW

在linux下编译FFmpeg很方便，但在Windows下还是有不少坑的，试了下使用Windows+MinGW编译FFmpeg，过程还是挺顺利的。记录一下(这里针对的平台是Android)。

NDK下载

点击这里直接下载安装程序，安装过程时长视网速而定。
